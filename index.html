I developed a cross-platform mobile application using Flutter and Dart, with Firebase as the backend. The app is designed to help users stay informed about upcoming events by providing real-time access to schedules, timings, and event details.

This project reflects my ability to build scalable, user-centric solutions that solve real-world problems. It’s especially useful for communities, campuses, or organizations that host frequent events and need a reliable way to communicate updates.
